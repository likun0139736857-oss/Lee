# 05 决策记录与思想路径

## 可观察的内容选择

1. **从自我发展切入**：先用《自我发展心理学50讲》建立行为、思维、关系、转折期、人生地图的总框架。
2. **把课程转化为图书**：《了不起的我》把课程中的工具和咨询故事大众化。
3. **进入亲密关系主题**：《爱，需要学习》和《亲密关系30讲》把“关系如何塑造自我”作为重点。
4. **把转变单独展开**：《自我转变50讲》和《走出黑森林》说明他把“转折期”看成足够重要、需要独立成体系的主题。
5. **进入家庭系统**：《家庭关系21讲》和2026年《中国式家庭》说明其关注点从个体和伴侣关系扩展到家庭治疗现场。

## 决策逻辑推断

陈海贤的作品路线不是随机选题，而是沿着“个体如何改变 -> 关系如何改变个体 -> 家庭如何塑造关系 -> 转折期如何重塑自我”的路径推进。

## 言行一致性

其作品反复强调实践、练习和新经验；课程和图书结构也普遍包含练习、行动工具、问答和案例，形式与观点一致。

## 需要保留的边界

本研究无法验证其私人职业选择、临床个案处理和未公开决策，只能依据公开作品路线进行推断。
